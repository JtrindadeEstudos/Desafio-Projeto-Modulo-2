# Desafio de Projeto – Site de Clínica Médica

Projeto desenvolvido como parte do curso **Formação HTML Web Developer** da **DIO**.

O objetivo deste desafio é criar um site simples para uma **clínica médica**, utilizando apenas **HTML**, seguindo a estrutura proposta no módulo e aplicando os conhecimentos aprendidos sobre navegação entre páginas, formulários, tabelas e organização de conteúdo.

## Estrutura do site
O site é composto pelas seguintes páginas:
- Página Principal
- Sobre a Clínica
- Horário de Atendimento
- Contato

## Tecnologias utilizadas
- HTML5
